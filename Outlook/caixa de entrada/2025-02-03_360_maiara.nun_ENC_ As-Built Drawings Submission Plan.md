Regarding the formatting requirements for CHESF as-built drawings, I have outlined the current understanding below based on the information available. Please note that these are for reference only, and if there are any uncertainties or additional requirements,

Current Understanding of CHESF As-Built Drawing Requirements:

1\. Modify the drawings based on the on-site change information, highlight the changes, and clearly mark the revised content.  

2\. Update the "Description" in the drawing frame to: "AS BUILT".  

3\. Update the CHESF version number in the drawing frame to the next sequential number following the previous version.  

4\. Change the "Issue purpose" in the drawing frame to: "PI" (Para Informação).  

5\. Upload the as-built drawings to the same folder as the construction drawings on the SGD platform, and fill in the GRD as: "AS BUILT".  

6\. Create and share a cloud storage link containing both the as-built drawings and the marked-up construction drawings. The link should include all project files in DWG and PDF formats, organized by discipline, and send it via email to the designated CHESF email address.  

7\. Print all SPCS and telecommunications as-built drawings in A3 format and deliver them to their respective installation locations. Only one hard copy is required (no hard copies are needed for other disciplines).  

Could you please review the attached change information and provide a submission plan for the as-built drawings for PWC’s review? Your prompt feedback would be greatly appreciated to ensure a smooth and timely completion of this phase.
